# Children
## Instructions
create a component `<Parent></Parent>` that will take the following HTML element as its children:

```html
  <div>Child 0...</div>
  <div>Child 1...</div>
  <div>Child 2...</div>
  <div>Child 3...</div>
```

render the component in `index.js` file.

**⚠️ your are not allowed to pass the element through a prop!**
